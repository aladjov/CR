About This Tutorial

This tutorial demonstrates the Customer Retention ML Framework using a synthetic retail dataset (30,801 customers). Each notebook walks through a stage of the ML pipeline, from data exploration to production validation.

Features demonstrated:

  • Automatic data profiling and type detection
  • Temporal scenario detection and point-in-time splits
  • Data quality assessment (99.7/100 score)
  • Feature engineering and capacity analysis
  • Model training: Logistic Regression, Random Forest, XGBoost
  • SHAP interpretability and feature importance
  • Distribution drift analysis and production validation

Key Lesson: Simpler models (Logistic Regression) can be more robust to distribution drift than complex models (XGBoost).
